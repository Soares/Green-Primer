Class Window

django.db.models.Model --+
                         |
                        Window

A type of window. Has a size, keeps track of how many times it is used in each layout. Because each floor will post the window counts separately, we have to keep window counts in their own tables and sum them when asked. This allows us to easily avoid race conditions, and this is why there's a 'WindowCount' model below.
